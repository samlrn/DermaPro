# DermaPro

DermaPro is a web-first Ionic React app for an on-demand dermatology staffing marketplace. The repo is set up for one shared codebase that can ship as a responsive web app and later wrap into native iOS and Android apps through Capacitor.

## Product Direction

The MVP focuses on the core booking loop:

1. A clinic posts an open dermatology shift.
2. Dermatologists discover, filter, save, and apply for shifts.
3. The clinic reviews applicants and confirms a booking.
4. The dermatologist accepts, messaging stays tied to the shift, and payment moves through Stripe.

The first screens in this repo map to that workflow: dashboard, shift discovery, bookings/payments, messaging, and profile/credentialing.

## Tech Stack

- Ionic React for mobile-ready UI components
- Vite and TypeScript for the web build
- Capacitor for iOS and Android project generation
- React Router through `@ionic/react-router`

## Getting Started

```bash
npm install
npm run dev
```

Build the web app:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Mobile Workflow

The repo is Capacitor-ready. After a successful web build, sync native projects:

```bash
npm run build
npm run cap:sync
```

Open native projects:

```bash
npm run cap:ios
npm run cap:android
```

Native folders are generated by Capacitor and should be committed once the team is ready to maintain platform-specific settings, signing, icons, splash screens, and permissions.
